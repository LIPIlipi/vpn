# 大概步骤
前提：安装openvpn所需要的依赖包，并且在windows里安装客户端
一、关闭防火墙 

二、开启路由转发 

三、配置server.conf 

四、生成证书，将证书放在server.conf特定路径中 

五、将证书复制打包放在windowsopenvpn客户端的配置文档中 

六、配置客户端client的配置文件，与server配置要相同才能链接
